# AtoM Framework

Laravel Query Builder integration for Access to Memory (AtoM) 2.10 with Extension Management System.

## Overview

- **Laravel Query Builder** (Illuminate\Database) integrated into AtoM's Symfony 1.4
- **Extension Manager** for installing, enabling, and managing AtoM extensions
- **Helper Classes** for common functionality
- **Repository/Service Pattern** for clean data access

## Requirements

- AtoM 2.10.x
- PHP 8.3+
- MySQL 8.0+
