Installation Methods

APT Repository
Direct Download
One-Liner

Add the APT Repository (one-time setup)

# Add the GPG key
curl -fsSL https://archiveheritagegroup.github.io/atom-framework/gpg.key | sudo gpg --dearmor -o /usr/share/keyrings/atom-heratio.gpg

# Add the repository
echo "deb [signed-by=/usr/share/keyrings/atom-heratio.gpg] https://archiveheritagegroup.github.io/atom-framework stable main" | sudo tee /etc/apt/sources.list.d/atom-heratio.list

# Update and install
sudo apt update
sudo apt install atom-heratio

The installer will launch an interactive setup wizard to configure your database, web server, and plugins.

Download and Install Directly

# Download the package
wget https://github.com/ArchiveHeritageGroup/atom-framework/releases/latest/download/atom-heratio_{{HERATIO_VERSION}}-1_all.deb

# Install (will resolve dependencies automatically)
sudo apt install ./atom-heratio_{{HERATIO_VERSION}}-1_all.deb

Quick Install Script

curl -fsSL https://archiveheritagegroup.github.io/atom-framework/install.sh | sudo bash

This adds the APT repository. After running, install with: sudo apt install atom-heratio
